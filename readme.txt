Este proyecto es un rediseño del Sistema de Gestión Académica (SGA) de la Universidad Técnica de Manabí (UTM). Se enfocó en mejorar la experiencia de usuario, creando una interfaz moderna, clara y adaptada a dispositivos móviles.

Tecnologías usadas
HTML5

CSS3 (avanzado: variables CSS, Flexbox, media queries, pseudo-elementos)

Estructura del proyecto
index.html: Archivo principal con la estructura semántica del sitio.

style.css: Archivo con todos los estilos, responsivos y modernos.

Funcionalidades principales
Menú de navegación con submenús desplegables usando solo CSS.

Diseño responsivo con menú hamburguesa para móviles.

Sección bienvenida con imagen de fondo y texto destacado.

Distribución clara y ordenada del contenido principal.

Cómo usar
Abre el archivo index.html en cualquier navegador moderno.

Navega por el menú para ver las categorías y submenús.

Cambia el tamaño de la ventana para probar la responsividad.

Notas

El diseño es modular y fácilmente adaptable para otros proyectos similares.
